FSDF Linked Data APIs

This is the homepage of the Foundational Spatial Data Framework (FSDF) Project's Linked Data Application Programming Interfaces (APIs).

This is one API providing access to multiple spatial datasets and vocabularies and there are other APIs similar to this one too:

  • This API - Spatial
    • Electrical Infrastructure
    • Facilities
    • Floods
    • Place Names
  • This API - Vocabularies
    • Multiple FSDF vocabularies
  • ASGS API
    • ASGS Edition 3
  • Geofabric API
    • Geofabric Version 3
  • G-NAF API
    • G-NAF, Latest public version

These APIs are the point-of-truth for their content and persistent identifiers for their content will resolve to them.

Additionally there is a combined API that contains a copy of all of the above, aggregated together for ease of cross-querying:

  • Cache API
    • All datasets
