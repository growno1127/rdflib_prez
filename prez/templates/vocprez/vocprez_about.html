About VocPrez

VocPrez is a read-only web delivery system - web pages and API - for Simple Knowledge Organization System (SKOS)-formulated RDF vocabularies. It complies with Content Negotiation by Profile.

Introduction

This is an instance of the new version of VocPrez - VocPrez 3. It is a part of a Linked Data presentation API called Prez. This instance is the first demo version of Prez.

The previous version of VocPrez is used by:

  • GeoScience Australia
  • Geological Survey of Queensland
  • Department of Agriculture, Water & Environment
  • CSIRO
  • British Oceanographic Data Centre
