About VocPrez

VocPrez is a read-only web delivery system - web pages and API - for Simple Knowledge Organization System (SKOS)-formulated RDF vocabularies. It complies with Content Negotiation by Profile.

Many instances of VocPrez are public and in production. See the Prez documentation for details.

Use

The basic structure of VocPrez is: 

        /                               - API Home

        /vocab                          - list of Vocabularies

        /vocab/{id}                     - a vocabulary

        /dataset/{id}/{concept-id}      - Concept within a vocabulary

Navigation to any of those endpoints delivers content in HTML, for humans, and multiple RDF formats, for machines.

This API provides alternate profiles of all content, so a the information is resented according to several different data models.

There are also convenience endpoints for searching and applying basic text or more advanced SPARQL queries.

SpacePrez' data is all natively stored in RDF graph form, according to the VocPub profile of SKOS and, optionally, according to other data models too, if implemented.
